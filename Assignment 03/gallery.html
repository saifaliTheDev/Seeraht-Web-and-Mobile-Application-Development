HTML5: Figure and Figure Caption

<figure></figure>

Images often comes with captions. HTML5 has introduced a new element to contain images and their captions so that the two were associated.
You can have more than one image inside the element as long as they all share the same caption.

<figcaption></figcaption>

The element has been added to HTML5 in order to allow web page authors to add a caption to an image.
Before the elements were created there was no way to associate an element with its caption.
Older browsers don't understand HTML5 elements simply ignore the new elements and display the content of them.
